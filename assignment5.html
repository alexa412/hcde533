Getting Meshy

Inspiration

I'm obsessed with dachshunds (see my family's dachshund, Winnie, below), so I found a dachshund STL on Thingiverse. I thought it would be fun to put a hat on the dachshund. So, I found a couple of different options, and ultimately went with a Harry Potter sorting hat! (sources linked at the end)

Process

I imported both STL meshes into Rhino. I had to:

  • Rotate the sorting hat mesh
  • Manually scale down the sorting hat
  • Use the QuadRemesh command on both meshes
  • Position the sorting hat mesh over the dachshund mesh
  • Rotate and scale the sorting hat mesh until it looked correct
  • MeshBooleanUnion to combine the meshes
This allowed me to create a single closed mesh, suitable for 3D printing (with supports).

Santa Hat

Just for fun, I repeated the same process with a Santa hat.

Modifying Meshes

I also spent some time messing around with the final mesh using the PointsOn command and manually dragging points to change the shape, although it ended up looking a bit scary.

Lamp Design

Idea

My lampshade idea is using translucent laser cut acrylic chains, inspired by these earrings. So, to hold the chains, I came up with an idea for a 3D printed base. The lamp innards fit inside of the base. The base slips onto the innards through a slot, which has clearance for the cord. Once slipped on, the base slides down onto the lamp innards. Ideally, this lamp would hang from somewhere by the cord. The base kind of looks like a fancy cake stand, I suppose. The width of the scalloped portion of the base is about 6 inches.

My initial idea was to do a cantilever snap fit design to hold the innards with a similar shade, but I pivoted to this design after having issues with 3D printing a snap fit holder. My design is specifically designed with some simpler geometry to allow for smoother 3D printing with minimal supports required. The base will be printed upside down. To save on material, I might make "top" or decoration for the base out of paper or something else.

Design Part 1 - The Base

First, I created a general shape for the base by measuring the outer dimensions of the lamp innards. I created a basic closed curve and revolved it to create the solid.

Design Part 2 - The Scalloped Edge

Unsurprisingly, there aren't great tutorials online for scalloped shapes and edges. I was all on my own for this part. Shout out to Junchao for walking me through doing this in Rhino so that I could recreate it in Grasshopper! Because I wasn't sure how big I wanted the scallops to be and how many I wanted on the edge, I decided to use Grasshopper for parametric modeling. I created the scalloped edge by (1) creating the scallop shape with a hole for the acrylic chains, (2) creating a linear array of scallops, (3) making a circle that matches the outer diameter of the scalloped edge, and (4) flowing the linear array of scallops onto the curve to create a circle of scallops. I used number sliders throughout this process to test different sizes.

I then baked the circle of scallops into Rhino, and used the BooleanUnion command to join the base to the scallops to create my scalloped lamp base. I added fillets to the edges, as well.


Design Part 3 - Cord Slot

To create a slot for the lamp cord to slip into the base, I created a 5.1mm wide extruded rectangle. I used the BooleanDifference command to create the slot, and then added some fillets. I'm going to print out the base later this week!

Design Part 4 - Acrylic Chain Shape

For now, I designed a few simple shape in Rhino for my acrylic chains. I'll be purchasing acrylic this week from TAP Plastics and laser cutting about 300 chain pieces at the MILL with the acrylic (12 chains per link, 25 links total).
